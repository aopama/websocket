# webchat

## 功能

- [x] 注册登录功能
- [x] 聊天功能 
- [x] 查看历史记录 
- [x] 多个聊天室 
- [x] 图片发送
- [x] 图片预览
- [x] 消息未读
- [x] 断线重连
- [x] 发表动态
- [x] 评论动态
- [x] 动态详情
- [x] 点赞动态
- [x] 点赞评论
- [x] 收藏动态
- [x] 个人中心
- [x] 后续开发

## 启动项目

Dev环境: MongoDB、Node 8.5.0+、Npm 5.3.0+

Prod环境: Redis、MongoDB、Node 8.5.0+、Npm 5.3.0+

```
npm install -----安装依赖

npm run dev -----运行

```

## 打包

```
npm run build 

npm run prod

```
